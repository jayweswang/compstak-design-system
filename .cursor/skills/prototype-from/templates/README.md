# Prototype Templates

This directory contains reusable HTML prototype templates that follow the Compstak design system.

## Using Templates

Templates are standalone HTML files that can be copied and customized for specific prototyping needs. Each template includes:
- Design system tokens
- Interactive component behaviors
- Responsive layouts
- Accessibility features

## Available Templates

Currently, this directory is empty. Templates need to be created following the guidelines in [TEMPLATES.md](../TEMPLATES.md).

## Creating Templates

See [TEMPLATES.md](../TEMPLATES.md) for detailed instructions on creating new templates.

Priority templates to build:
1. Modal patterns (confirmation, form, info)
2. Form patterns (basic, multi-step, search)
3. Card layouts (grid, list, dashboard)
4. Navigation patterns (top nav, side nav, tabs)
5. Data display (table, list, timeline)

## Template Standards

All templates must:
- Use design system tokens from `design-system.index.mdc`
- Follow component behaviors from `component-behaviors.md`
- Include customization markers in HTML comments
- Be fully responsive
- Include accessibility features
- Pass design-review validation
